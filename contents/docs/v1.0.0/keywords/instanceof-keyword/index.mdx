# <I18nEnglish>instanceof Operator</I18nEnglish>
# <I18nBangla>উদাহরণ_হিসেবে অপারেটর</I18nBangla>
# <I18nBanglish>instanceof Operator</I18nBanglish>

<I18nEnglish>
**`instanceof`** tests whether the **prototype chain** of a value contains the `prototype` of a given **constructor** (function or class).  
It returns **true/false** and works only for **objects** (not primitives), unless custom logic is provided via **`Symbol.hasInstance`**.
</I18nEnglish>

<I18nBangla>
**`উদাহরণ_হিসেবে`** অপারেটর কোনো মানের **প্রোটোটাইপ চেইন**‑এ নির্দিষ্ট **কন্সট্রাক্টর**‑এর `prototype` আছে কি না তা যাচাই করে।  
এটি **সত্য/মিথ্যা** দেয় এবং কেবল **অবজেক্ট**‑এর জন্য নির্ভরযোগ্য (প্রিমিটিভে নয়), তবে **`Symbol.hasInstance`** দিয়ে কাস্টম নিয়ম দেওয়া যায়।
</I18nBangla>

<I18nBanglish>
**`udahoron_hisebe`** diye dekha hoy value‑r **prototype chain** e deya **constructor**‑er `prototype` ache kina.  
**true/false** dey; **object** e kaj kore (primitive e na), kintu **`Symbol.hasInstance`** diye custom rule deya jay.
</I18nBanglish>

## Notes

<I18nEnglish>
- `instanceof` walks the **prototype chain**; it does **not** compare constructors directly.  
- Works **only** on objects. For primitives, prefer `typeof`. Wrapper objects (`new String("x")`) are generally discouraged.  
- **Cross‑realm** (iframes, vm contexts) mean different constructors; `x instanceof Array` may be `false`. Prefer **`Array.isArray(x)`** for arrays.  
- You can customize behavior via `static [Symbol.hasInstance](value)` on a constructor.  
- For structural checks, use **duck typing** or schema validation instead of `instanceof`.
</I18nEnglish>

<I18nBangla>
- `উদাহরণ_হিসেবে` **প্রোটোটাইপ চেইন** ঘুরে দেখে; কন্সট্রাক্টর সরাসরি তুলনা করে না।  
- কেবল **অবজেক্ট**‑এ নির্ভরযোগ্য; প্রিমিটিভের জন্য `ধরন` (`typeof`) ব্যবহার করুন। র‍্যাপার অবজেক্ট (`new String`) সাধারণত এড়ান।  
- **ক্রস‑রিয়েল্ম** (iframe/vm)‑এ আলাদা কন্সট্রাক্টর থাকে—তাই `x উদাহরণ_হিসেবে Array` মিথ্যা হতে পারে; অ্যারের জন্য **`Array.isArray(x)`** ভালো।  
- কন্সট্রাক্টরে `static [Symbol.hasInstance](value)` লিখে আচরণ কাস্টমাইজ করা যায়।  
- কাঠামোগত যাচাইয়ের জন্য `instanceof`‑এর বদলে **ডাক টাইপিং/স্কিমা ভ্যালিডেশন** ব্যবহার করুন।
</I18nBangla>

<I18nBanglish>
- `udahoron_hisebe` prototype chain ghure dekhe; constructor direct compare kore na.
