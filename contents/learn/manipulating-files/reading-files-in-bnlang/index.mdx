<I18nEnglish>
Reading files is a core operation in most applications.  
Bnlang provides both **synchronous** and **asynchronous** methods for reading files.  
Synchronous methods block the program until the file is read, while asynchronous methods allow the program to continue running while the file is being processed.
</I18nEnglish>

<I18nBangla>
ফাইল পড়া বেশিরভাগ অ্যাপ্লিকেশনের একটি মূল অপারেশন।  
Bnlang ফাইল পড়ার জন্য **সিঙ্ক্রোনাস** এবং **অ্যাসিঙ্ক্রোনাস** উভয় পদ্ধতি প্রদান করে।  
সিঙ্ক্রোনাস পদ্ধতিতে ফাইল পড়া শেষ না হওয়া পর্যন্ত প্রোগ্রাম ব্লক থাকে, আর অ্যাসিঙ্ক্রোনাস পদ্ধতিতে ফাইল প্রসেসিং চলার সময়ও প্রোগ্রাম চালু থাকে।
</I18nBangla>

<I18nBanglish>

## Example: Asynchronous Read

```js
const fs = require("fs");

fs.readFile("example.txt", "utf-8", (err, data) => {
  if (err) {
    console.error("Error reading file:", err);
    return;
  }
  console.log("File content:", data);
});
```

## Best Practices

<I18nEnglish>
- Prefer asynchronous methods in production for scalability.  
- Use synchronous methods only for small scripts or quick utilities.  
- Always handle errors properly when reading files.  
- Use `utf-8` encoding for text files unless binary data is required.
</I18nEnglish>

<I18nBangla>
- প্রোডাকশনে স্কেলেবিলিটির জন্য অ্যাসিঙ্ক্রোনাস মেথড ব্যবহার করুন।  
- ছোট স্ক্রিপ্ট বা ইউটিলিটির জন্য কেবল সিঙ্ক্রোনাস মেথড ব্যবহার করুন।  
- ফাইল পড়ার সময় সর্বদা ত্রুটি সঠিকভাবে হ্যান্ডল করুন।  
- টেক্সট ফাইলের জন্য `utf-8` এনকোডিং ব্যবহার করুন, যদি না বাইনারি ডেটা প্রয়োজন হয়।
</I18nBangla>

<I18nBanglish>
- Production e scalability er jonno asynchronous method use korun.  
- Choto script ba utility er jonno sudhu synchronous method use korun.  
- File read korar shomoy always error properly handle korun.  
- Text file er jonno `utf-8` encoding use korun, jodi binary data na lage.
</I18nBanglish>
