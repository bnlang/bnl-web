# <I18nEnglish>function Keyword</I18nEnglish>
# <I18nBangla>ফাংশন কীওয়ার্ড</I18nBangla>


<I18nEnglish>
Functions are reusable blocks of code.  
Define them with **`function`** (declaration or expression) or with **arrow functions** `() => ...`.  
They can take parameters, return values, and be **async**.
</I18nEnglish>

<I18nBangla>
ফাংশন হলো পুনঃব্যবহারযোগ্য কোডের ব্লক।  
**`ফাংশন`/`function`** দিয়ে ঘোষণা বা এক্সপ্রেশন হিসেবে লেখা যায়, এবং **অ্যারো ফাংশন** `() => ...` হিসেবেও লেখা যায়।  
প্যারামিটার নিতে পারে, **ফেরত** দিতে পারে, এবং **অসমলয়**ও হতে পারে।
</I18nBangla>

## Syntax

<I18nEnglish>

```bnl
// Declaration (hoisted)
function add(a, b) {
  return a + b;
}

// Function expression
const mul = function (a, b) {
  return a * b;
};

// Arrow function
const sub = (a, b) => a - b;

// Default & rest parameters
function greet(name = "Guest", ...tags) {
  console.log("Hello", name, tags);
}

// Async function
async function load(url) {
  const res = await fetch(url);
  if (!res.ok) throw new Error("Bad response");
  return await res.text();
}
```
</I18nEnglish>

<I18nBangla>

```bnl
// ঘোষণা (hoisted)

  ফেরত ক + খ;

const গুণ = function (ক, খ) {
  ফেরত ক * খ;

// অ্যারো ফাংশন
const বিয়োগ = (ক, খ) => ক - খ;

// ডিফল্ট ও রেস্ট প্যারামিটার
ফাংশন শুভেচ্ছা(নাম = "অতিথি", ...ট্যাগ) {
  কনসোল.লগ("হ্যালো", নাম, ট্যাগ);

অসমলয় ফাংশন লোড(url) {
  const res = অপেক্ষা fetch(url);
  যদি (!res.ok) নিক্ষেপ new Error("খারাপ রেসপন্স");
  ফেরত অপেক্ষা res.text();
}
```
</I18nBangla>

## Notes

<I18nEnglish>
- **Declarations** are hoisted; **expressions/arrow** are not (bindings exist but `let/const` are in the **TDZ**).  
- **Arrow functions** don’t have their own `this`, `arguments`, or `prototype`; avoid using them as constructors or methods needing dynamic `this`.  
- **Default parameters** evaluate at call time; **rest** collects remaining args into an array.  
- Prefer **early returns** to reduce nesting; name functions for clearer stack traces.  
- Deep recursion may overflow—convert to loops or use stacks where necessary.  
- Keep language style consistent per file (English/Bangla).
</I18nEnglish>

<I18nBangla>
- **ঘোষণা** hoisted হয়; **এক্সপ্রেশন/অ্যারো** hoisted নয় (`let/const` থাকলে **TDZ** প্রযোজ্য)।  
- **অ্যারো ফাংশন**‑এ নিজস্ব `this`/`arguments`/`prototype` থাকে না; কনস্ট্রাক্টর বা ডাইনামিক `this`‑কেন্দ্রিক মেথডে ব্যবহার এড়ান।  
- **ডিফল্ট প্যারামিটার** কল‑টাইমে হিসাব হয়; **রেস্ট** বাকি আর্গুমেন্ট অ্যারেতে নেয়।  
- নেস্টিং কমাতে **আগেভাগে ফেরত** দিন; ফাংশনের নাম দিন যেন স্ট্যাক ট্রেস পরিষ্কার থাকে।  
- গভীর রিকার্শনে স্ট্যাক ওভারফ্লো হতে পারে—লুপ/স্ট্যাক ব্যবহার করুন।  
- প্রতি ফাইলে একটিই ভাষা স্টাইল অনুসরণ করুন।
</I18nBangla>
