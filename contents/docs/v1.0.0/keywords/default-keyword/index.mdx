# <I18nEnglish>default Keyword</I18nEnglish>
# <I18nBangla>অন্যথায় কীওয়ার্ড</I18nBangla>
# <I18nBanglish>default Keyword</I18nBanglish>

<I18nEnglish>
The **`default`** branch in a `switch` runs when none of the `case` labels match.  
It is optional but recommended to handle unexpected values.
</I18nEnglish>

<I18nBangla>
`বিকল্প` (switch) স্টেটমেন্টে **`অন্যথায়`** ব্লক চালানো হয় যখন কোনো **`অবস্থা`** (case) মেলে না।  
এটি ঐচ্ছিক, তবে অপ্রত্যাশিত মান হ্যান্ডল করতে ব্যবহার করা ভালো।
</I18nBangla>

<I18nBanglish>
`bikolpo` (switch) e **`onnothay`** cholbe jokhon kono **`obostha`** (case) match kore na.  
Eta optional, kintu unexpected value handle korte dewa ucit.
</I18nBanglish>

## Syntax

<I18nEnglish>

```bnl
switch (expr) {
  case v1:
    // ...
    break;
  // ...
  default:
    // fallback logic
}
```
</I18nEnglish>

<I18nBangla>

```bnl
বিকল্প (অভিব্যক্তি) {
  অবস্থা মান১:
    // ...
    থামুন;
  // ...
  অন্যথায়:
    // বিকল্প/ফলব্যাক লজিক
}
```
</I18nBangla>

<I18nBanglish>

```bnl
bikolpo (expression) {
  obostha man1:
    // ...
    thamun;
  // ...
  onnothay:
    // fallback logic
}
```
</I18nBanglish>

const code = "X";

## Notes

<I18nEnglish>
- Only one **`default`** is allowed per switch.  
- `default` can appear anywhere; execution jumps there only when no `case` matches.  
- If you place `default` before other cases and omit `break`, execution will **fall through** to the next case.
</I18nEnglish>

<I18nBangla>
- একটি `বিকল্প` ব্লকে মাত্র একটি **`অন্যথায়`** থাকতে পারে।  
- **`অন্যথায়`** যেকোনো স্থানে রাখা যায়; কোনো `অবস্থা` না মিললে তবেই চালু হবে।  
- **`অন্যথায়`** এর পরে `থামুন` না দিলে পরের `অবস্থা`তে **fall-through** ঘটবে।
</I18nBangla>

<I18nBanglish>
- Protiti `bikolpo` te sudhu ekta **`onnothay`** thake.  
- **`onnothay`** je kono jaygay thakte pare; kono `obostha` match na korle tokhon cholbe.  
- `onnothay` er pore `thamun` na dile porer `obostha` te **fall-through** hobe.
</I18nBanglish>
