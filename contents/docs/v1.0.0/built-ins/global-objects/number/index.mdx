# <I18nEnglish>Number</I18nEnglish>
# <I18nBangla>সংখ্যা</I18nBangla>


<I18nEnglish>
Bnlang provides the number constructor and helpers under **`Number`**, **`সংখ্যা`**, and **`Sonkha`**.  
It contains constants, type-checking methods, parsing utilities, and prototype formatting methods.
</I18nEnglish>

<I18nBangla>
Bnlang এ **`Number`**, **`সংখ্যা`**, ও **`Sonkha`** ব্যবহার করা যায়।  
এতে ধ্রুবক, সংখ্যা চেক করার মেথড, পার্সিং ফাংশন ও ফরম্যাটিং মেথড রয়েছে।
</I18nBangla>

## Static Methods

<I18nEnglish>
- `Number.isFinite(x)`, `Number.isInteger(x)`, `Number.isNaN(x)`, `Number.isSafeInteger(x)`  
- `Number.parseInt(str)`, `Number.parseFloat(str)`
</I18nEnglish>

<I18nBangla>
- `সংখ্যা.সসীম_কি(x)`, `সংখ্যা.পূর্ণসংখ্যা_কি(x)`, `সংখ্যা.সংখ্যা_নয়_কি(x)`, `সংখ্যা.নিরাপদ_পূর্ণসংখ্যা_কি(x)`
