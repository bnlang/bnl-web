# <I18nEnglish>WeakSet</I18nEnglish>
# <I18nBangla>দুর্বল_সমষ্টি</I18nBangla>
# <I18nBanglish>WeakSet</I18nBanglish>

<I18nEnglish>
Use **`WeakSet`**, **`দুর্বল_সমষ্টি`**, or **`DurbolSomoshti`** to keep a set of **objects only** held weakly (eligible for GC). It supports just **`add`**, **`has`**, and **`delete`**. It is **not iterable** and has **no `size`**.
</I18nEnglish>

<I18nBangla>
**`দুর্বল_সমষ্টি`** অবজেক্ট-শুধু সংগ্রহ; মানগুলো দুর্বলভাবে ধরা থাকে, তাই অন্য রেফারেন্স না থাকলে GC হতে পারে। এখানে কেবল **`যোগ`**, **`আছে_কি`**, **`মুছুন`** আছে; এটি **ইটারেবল নয়** এবং **`size` নেই**।
</I18nBangla>

<I18nBanglish>
**`DurbolSomoshti`** holo object-only collection; values weak vabe dhora thake (GC hote pare). Shudhu **`jog`**, **`hasVal`**, **`deleteVal`** ache; **iterable na** ebong **`size` nai**.
</I18nBanglish>

## Notes

<I18nEnglish>
- Only **objects** can be stored; primitives (numbers, strings, etc.) are not allowed.
- Not enumerable: no `size`, `keys()`, `values()`, `entries()`, or `forEach()`.
</I18nEnglish>

<I18nBangla>
- কেবল **অবজেক্ট** রাখা যায়; প্রিমিটিভ (সংখ্যা, স্ট্রিং ইত্যাদি) রাখা যায় না।
- **ইটারেবল নয়**: `size`, `keys()`, `values()`, `entries()`, `forEach()` নেই।
</I18nBangla>

<I18nBanglish>
- Shudhu **object** rakhte paren; primitive (number, string, etc.) allowed na.
- **Enumerable na**: `size`, `keys()`, `values()`, `entries()`, `forEach()` nei.
</I18nBanglish>
