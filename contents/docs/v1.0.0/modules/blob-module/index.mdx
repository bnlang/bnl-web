# <I18nEnglish>blob</I18nEnglish>
# <I18nBangla>blob</I18nBangla>


<I18nEnglish>
The **`Blob`** class represents immutable raw binary data. Blobs are commonly used for file-like objects, uploads, and binary manipulation.
</I18nEnglish>

<I18nBangla>
**`Blob`** ক্লাস ইমিউটেবল কাঁচা বাইনারি ডেটা বোঝায়। এটি ফাইল-জাতীয় অবজেক্ট, আপলোড ও বাইনারি ম্যানিপুলেশনে ব্যবহৃত হয়।
</I18nBangla>

## API Surface

<I18nEnglish>

### Class: Blob
- **`new Blob(sources[, options])`** — create blob from array of parts.  
  - Parts: strings, BufferSource, Blob.  
  - Options: `{ type: "mime/type" }`.  

### Properties
- **`blob.size`** — size in bytes.  
- **`blob.type`** — MIME type string.  

### Methods
- **`blob.slice([start], [end], [type])`** — return a new Blob view.  
- **`blob.arrayBuffer()`** — resolve to ArrayBuffer.  
- **`blob.text()`** — resolve to string.  
- **`blob.stream()`** — readable stream of blob data.  

</I18nEnglish>

<I18nBangla>
### প্রপার্টি ও মেথড
- **`size`**, **`type`** — ব্লবের আকার ও টাইপ।  
- **`slice`**, **`arrayBuffer`**, **`text`**, **`stream`** — ডেটা অ্যাক্সেস।  
</I18nBangla>

const txt = await b.text();

## Notes

<I18nEnglish>
- Blobs are **immutable**; operations return new Blobs.  
- MIME type is advisory; consumers may ignore it.  
- Use **`Blob`** with **FormData** for file uploads.  
- For random access binary edits, use **Buffer** instead.  
</I18nEnglish>

<I18nBangla>
- Blob ইমিউটেবল; অপারেশনে নতুন Blob ফেরত দেয়।  
- MIME টাইপ শুধু তথ্য দেয়, ক্লায়েন্ট উপেক্ষা করতে পারে।  
- আপলোডের জন্য FormData এর সাথে Blob ব্যবহার করুন।  
- র‍্যান্ডম অ্যাক্সেস এডিটের জন্য Buffer ব্যবহার করুন।
