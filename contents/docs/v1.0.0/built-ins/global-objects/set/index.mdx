# <I18nEnglish>Set</I18nEnglish>
# <I18nBangla>সমষ্টি</I18nBangla>


<I18nEnglish>
The **Set** object stores unique values. In Bnlang, you can also use **`সমষ্টি`** or **`Somoshti`**. A value in a Set may only occur once.
</I18nEnglish>

<I18nBangla>
**সমষ্টি** দিয়ে অনন্য মান রাখা যায়। প্রতিটি মান একবারই থাকতে পারবে।
</I18nBangla>

## Methods

<I18nEnglish>
- `add(value)` – Insert a value.
- `delete(value)` – Remove a value.
- `clear()` – Remove all values.
- `has(value)` – Check existence.
- `size` – Count values.
- Iteration: `keys()`, `values()`, `entries()`, `forEach()`.
</I18nEnglish>

<I18nBangla>
- `যোগ(মান)` – একটি মান যোগ করে।
- `মোছো(মান)` – একটি মান মুছে দেয়।
- `পরিষ্কার()` – সব মান মুছে দেয়।
- `আছে(মান)` – মান আছে কিনা দেখে।
- `আকার` – মোট মানের সংখ্যা।
- ইটারেশন: `চাবিগুলি()`, `মানগুলি()`, `জোড়াগুলি()`, `প্রতিটি_চালান()`।
</I18nBangla>
