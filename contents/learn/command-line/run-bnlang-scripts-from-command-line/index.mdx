# <I18nEnglish>Run Bnlang Scripts from Command Line</I18nEnglish>
# <I18nBangla>কমান্ড লাইন থেকে Bnlang স্ক্রিপ্ট চালানো</I18nBangla>


<I18nEnglish>
Bnlang programs can be executed directly from the command line using the **`bnl`** command.  
This allows developers to quickly test, debug, and run scripts without extra setup.
</I18nEnglish>

<I18nBangla>
**`bnl`** কমান্ড ব্যবহার করে সরাসরি কমান্ড লাইন থেকে Bnlang প্রোগ্রাম চালানো যায়।  
এর মাধ্যমে ডেভেলপাররা দ্রুত টেস্ট, ডিবাগ এবং স্ক্রিপ্ট চালাতে পারে।
</I18nBangla>

## Running with Arguments

<I18nEnglish>
You can pass command-line arguments to Bnlang scripts using `process.argv`.
</I18nEnglish>

<I18nBangla>
`process.argv` ব্যবহার করে Bnlang স্ক্রিপ্টে কমান্ড লাইন আর্গুমেন্ট পাঠানো যায়।
</I18nBangla>



<Tabs defaultValue="english" className="w-full">
  <TabsList>
    <TabsTrigger value="english">English</TabsTrigger>
    <TabsTrigger value="bangla">Bangla</TabsTrigger>
    
  </TabsList>

  <TabsContent value="english">

```js
// file: greet.bnl
const name = process.argv[2] || "Guest";
console.log("Hello,", name);
```

```sh
bnl greet.bnl Alice
# Output: Hello, Alice
```
  </TabsContent>

  <TabsContent value="bangla">

```bnl
// ফাইল: greet.bnl
const নাম = process.argv[2] || "অতিথি";
কনসোল.লগ("হ্যালো,", নাম);
```

```sh
bnl greet.bnl আলিস
# আউটপুট: হ্যালো, আলিস
```

## Best Practices

<I18nEnglish>
- Keep scripts small and modular.  
- Use meaningful file names.  
- Validate input when using arguments.  
- Organize scripts in a `scripts/` directory.
</I18nEnglish>

<I18nBangla>
- স্ক্রিপ্ট ছোট এবং মডুলার রাখুন।  
- অর্থবহ ফাইল নাম ব্যবহার করুন।  
- আর্গুমেন্ট ব্যবহার করলে ইনপুট যাচাই করুন।  
- স্ক্রিপ্টগুলো `scripts/` ডিরেক্টরিতে রাখুন।
</I18nBangla>
