# <I18nEnglish>Global Properties</I18nEnglish>
# <I18nBangla>গ্লোবাল প্রোপার্টি</I18nBangla>
# <I18nBanglish>Global Properties</I18nBanglish>

<I18nEnglish>
Bnlang provides some built-in global properties just like JavaScript:
- **`undefined`** → `অসংজ্ঞায়িত` / `asongyayito`  
- **`NaN`** → `সংখ্যা_নয়` / `sonkhaNoy`  
- **`Infinity`** → `অসীম` / `oshim`  
These are constants automatically available everywhere.
</I18nEnglish>

<I18nBangla>
Bnlang এ কিছু বিল্ট-ইন গ্লোবাল প্রোপার্টি আছে:
- **`অসংজ্ঞায়িত`** → `undefined`  
- **`সংখ্যা_নয়`** → `NaN`  
- **`অসীম`** → `Infinity`  
এগুলো সর্বত্র ব্যবহার করা যায়।
</I18nBangla>

<I18nBanglish>
Bnlang e kichu built-in global property ache:
- **`asongyayito`** → `undefined`  
- **`sonkhaNoy`** → `NaN`  
- **`oshim`** → `Infinity`  
Egulo sob jaigai use kora jay.
</I18nBanglish>
