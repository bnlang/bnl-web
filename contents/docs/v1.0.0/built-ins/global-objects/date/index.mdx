# <I18nEnglish>Date</I18nEnglish>
# <I18nBangla>তারিখ</I18nBangla>

<I18nEnglish>
Use the date/time API with any of these names: **`Date`**, **`তারিখ`**, **`Tarikh`**.
Create instances with `new Date()` / `new তারিখ()` / `new Tarikh()` and call the same operations by their English, Bangla, or Banglish forms.
</I18nEnglish>

<I18nBangla>
তারিখ/সময়ের API এই নামগুলোতে পাওয়া যায়: **`Date`**, **`তারিখ`**, **`Tarikh`**।
ইনস্ট্যান্স বানাতে `new Date()` / `new তারিখ()` / `new Tarikh()` ব্যবহার করুন এবং প্রয়োজনমতো ইংরেজি/বাংলা/Banglish নামে একই কাজ ডাকুন।
</I18nBangla>

## Static Methods

<I18nEnglish>
- Current timestamp: `Date.now()`, `তারিখ.এখন()`, `Tarikh.ekhon()`
- Parse string: `Date.parse(str)`, `তারিখ.বিশ্লেষণ(str)`, `Tarikh.bishleshon(str)`
- Build UTC: `Date.UTC(y, m, d, h, M, s, ms)`, `তারিখ.ইউটিসি(...)`, `Tarikh.utc(...)`
</I18nEnglish>

<I18nBangla>
- বর্তমান সময়স্ট্যাম্প: `তারিখ.এখন()` / `Date.now()` / `Tarikh.ekhon()`
- স্ট্রিং বিশ্লেষণ: `তারিখ.বিশ্লেষণ(str)` / `Date.parse(str)`
- ইউটিসি তৈরি: `তারিখ.ইউটিসি(...)` / `Date.UTC(...)` / `Tarikh.utc(...)`
</I18nBangla>

<I18nBanglish>
- Current timestamp: `Tarikh.ekhon()` / `Date.now()` / `তারিখ.এখন()`
- Parse string: `Tarikh.bishleshon(str)` / `Date.parse(str)`
- Make UTC: `Tarikh.utc(...)` / `Date.UTC(...)` / `তারিখ.ইউটিসি(...)`
</I18nBanglish>

## Instance Methods (Common)

<I18nEnglish>
- Get parts: `getFullYear()` / `getMonth()` / `getDate()` → Bangla: `বছর()` / `মাস()` / `তারিখ()` · Banglish: `bochor()` / `mas()` / `tarikh()`
- Set parts: `setFullYear()` / `setMonth()` / `setDate()` → Bangla: `বছর_স্থাপন()` / `মাস_স্থাপন()` / `তারিখ_স্থাপন()` · Banglish: `bochorSthapon()` / `masSthapon()` / `tarikhSthapon()`
- Time value: `getTime()` / `setTime()` → Bangla: `সময়()` / `সময়_স্থাপন()` · Banglish: `shomoy()` / `shomoySthapon()`
- Local/UTC: `toLocaleString()` / `toUTCString()` → Bangla: `স্থানীয়_অক্ষরমালায়_রুপান্তর()` / `ইউটিসি_অক্ষরমালায়_রুপান্তর()` · Banglish: `sthaniyoOkkhoromalayRupantor()` / `utcOkkhoromalayRupantor()`
</I18nEnglish>

<I18nBangla>
- অংশ পড়া: `বছর()`, `মাস()`, `তারিখ()` · `getFullYear()`, `getMonth()`, `getDate()`
- অংশ সেট: `বছর_স্থাপন()`, `মাস_স্থাপন()`, `তারিখ_স্থাপন()` · `setFullYear()`, `setMonth()`, `setDate()`
- সময় মান: `সময়()`, `সময়_স্থাপন()` · `getTime()`, `setTime()`
- লোকাল/ইউটিসি: `স্থানীয়_অক্ষরমালায়_রুপান্তর()`, `ইউটিসি_অক্ষরমালায়_রুপান্তর()` · `toLocaleString()`, `toUTCString()`
</I18nBangla>

<I18nBanglish>
- Part read: `bochor()`, `mas()`, `tarikh()` · `getFullYear()`, `getMonth()`, `getDate()`
- Part set: `bochorSthapon()`, `masSthapon()`, `tarikhSthapon()` · `setFullYear()`, `setMonth()`, `setDate()`
- Time value: `shomoy()`, `shomoySthapon()` · `getTime()`, `setTime()`
- Local/UTC: `sthaniyoOkkhoromalayRupantor()`, `utcOkkhoromalayRupantor()` · `toLocaleString()`, `toUTCString()`
</I18nBanglish>
