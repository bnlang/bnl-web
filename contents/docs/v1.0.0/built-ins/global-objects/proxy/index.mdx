# <I18nEnglish>Proxy</I18nEnglish>
# <I18nBangla>প্রক্সি</I18nBangla>


<I18nEnglish>
A **Proxy** (Bangla: **প্রক্সি**) wraps a target object and a handler to intercept operations like `get`, `set`, etc.

Static method: `Proxy.revocable(target, handler)` · Bangla: `প্রক্সি.প্রত্যাহারযোগ্য(target, handler)` · Banglish: `Proxy.prottaharjuggo(target, handler)`
</I18nEnglish>

<I18nBangla>
**প্রক্সি** দিয়ে একটি টার্গেট অবজেক্টকে মোড়ানো হয় এবং `get`, `set` ইত্যাদি অপারেশন ইন্টারসেপ্ট করা যায়।

স্ট্যাটিক মেথড: `প্রক্সি.প্রত্যাহারযোগ্য(target, handler)`
</I18nBangla>

const target = {};
const handler = {
  get: (obj, prop) => (prop in obj ? obj[prop] : "default")
};

const p = new Proxy(target, handler);
console.log(p.abc); // "default"

const { proxy, revoke } = Proxy.revocable(target, handler);
console.log(proxy.abc); // "default"
revoke();
