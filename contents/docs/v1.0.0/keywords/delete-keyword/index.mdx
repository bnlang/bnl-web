# <I18nEnglish>delete Operator</I18nEnglish>
# <I18nBangla>মুছুন অপারেটর</I18nBangla>
# <I18nBanglish>delete Operator</I18nBanglish>

<I18nEnglish>
The **`delete`** operator removes a **property** from an object.  
It returns **`true`** if the property was removed (or didn’t exist), and **`false`** for **non‑configurable** properties in non‑strict mode.  
In **strict mode**, attempting to delete a non‑configurable property or an unqualified identifier throws a **`TypeError`/`SyntaxError`**.
</I18nEnglish>

<I18nBangla>
**`মুছুন`** অপারেটর দিয়ে **অবজেক্টের প্রপার্টি** মুছে ফেলা হয়।  
প্রপার্টি মুছতে পারলে (বা প্রপার্টি না থাকলে) **`true`** ফেরত দেয়; **non‑configurable** হলে non‑strict‑এ **`false`**।  
**strict mode**‑এ non‑configurable বা সরাসরি আইডেন্টিফায়ার মুছতে গেলে **`TypeError`/`SyntaxError`** হয়।
</I18nBangla>

<I18nBanglish>
**`muchun`** (delete) object er **property** tule dey.  
Delete safol hole (ba property thakleo na) **`true`**, **non‑configurable** hole non‑strict e **`false`**.

</I18nBanglish>

## Notes

<I18nEnglish>
- `delete` only removes **properties**. For arrays, it removes the **slot** and leaves a **hole** (length unchanged); prefer **`.splice`** if you need to shift indices.  
- Returns `true` if the property doesn’t exist or is successfully removed; returns `false` for **non‑configurable** properties in non‑strict code.  
- In **strict mode**, deleting a **non‑configurable** property throws; deleting an **unqualified identifier** (e.g., `delete x`) is a **SyntaxError**.  
- Global `var`/function bindings are non‑configurable properties of the global object — they cannot be deleted. `let/const` are **lexical bindings** and are never deletable.  
- For **Map/Set**, use `.delete(key)` instead of `delete`.  
- Keep one language style (English/Bangla/Banglish) per file.
</I18nEnglish>

<I18nBangla>
- `মুছুন` কেবল **প্রপার্টি** মুছে; অ্যারেতে **slot** ফাঁকা হয়ে **hole** থাকে—ইনডেক্স সরাতে **`.splice`** ব্যবহার করুন।  
- প্রপার্টি না থাকলেও `true`; non‑strict‑এ **non‑configurable** হলে `false`।  
- **strict mode**‑এ non‑configurable মুছতে গেলে ত্রুটি; **সরাসরি আইডেন্টিফায়ার** (`মুছুন x`) মুছতে গেলে **SyntaxError**।  
- গ্লোবাল `var`/function গুলি গ্লোবাল অবজেক্টের **non‑configurable** প্রপার্টি—মোছা যায় না। `let/const` **lexical binding**, কখনোই deletable নয়।  
- **Map/Set**‑এ `delete` নয়—`.delete` ব্যবহার করুন।  
- প্রতি ফাইলে এক ভাষা স্টাইল রাখুন।
</I18nBangla>
