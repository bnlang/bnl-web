# <I18nEnglish>Bnlang File Stats</I18nEnglish>
# <I18nBangla>Bnlang ফাইল স্ট্যাটস</I18nBangla>


<I18nEnglish>
File statistics (or **file stats**) provide detailed information about a file, such as its size, type, creation date, and last modification date.  
In Bnlang, you can use built-in modules to get these details and make decisions in your program, such as checking if a path is a file or a directory.
</I18nEnglish>

<I18nBangla>
ফাইল পরিসংখ্যান (**ফাইল স্ট্যাটস**) একটি ফাইলের বিস্তারিত তথ্য দেয়, যেমন সাইজ, ধরন, তৈরির তারিখ এবং সর্বশেষ পরিবর্তনের তারিখ।  
Bnlang এ আপনি বিল্ট-ইন মডিউল ব্যবহার করে এই তথ্যগুলি পেতে পারেন এবং আপনার প্রোগ্রামে সিদ্ধান্ত নিতে পারেন, যেমন একটি পাথ ফাইল না ডিরেক্টরি তা যাচাই করা।
</I18nBangla>

```js
const fs = require("fs");

fs.stat("example.txt", (err, stats) => {
  if (err) {
    console.error("Error:", err);
    return;
  }
  console.log("Is file?", stats.isFile());
  console.log("Is directory?", stats.isDirectory());
  console.log("Size:", stats.size);
  console.log("Created:", stats.birthtime);
  console.log("Modified:", stats.mtime);
});
```

## Common Properties

<I18nEnglish>
- **isFile()** → Checks if the path is a file.  
- **isDirectory()** → Checks if the path is a directory.  
- **size** → Returns the file size in bytes.  
- **birthtime** → The date the file was created.  
- **mtime** → The date the file was last modified.
</I18nEnglish>

<I18nBangla>
- **isFile()** → যাচাই করে পাথটি ফাইল কিনা।  
- **isDirectory()** → যাচাই করে পাথটি ডিরেক্টরি কিনা।  
- **size** → ফাইলের সাইজ (বাইটে) দেয়।  
- **birthtime** → ফাইলটি কবে তৈরি হয়েছে তার তারিখ দেয়।  
- **mtime** → ফাইলটি সর্বশেষ কখন পরিবর্তিত হয়েছে তা জানায়।
</I18nBangla>

## Best Practices

<I18nEnglish>
- Always handle errors when reading file stats.  
- Use `isFile()` and `isDirectory()` before performing operations.  
- File sizes are in bytes, so convert them if needed (e.g., KB, MB).  
- Use stats to check last modification date for caching or versioning logic.
</I18nEnglish>

<I18nBangla>
- ফাইল স্ট্যাটস পড়ার সময় সবসময় ত্রুটি হ্যান্ডল করুন।  
- কোনো অপারেশন করার আগে `isFile()` এবং `isDirectory()` ব্যবহার করুন।  
- ফাইলের সাইজ বাইটে হয়, তাই প্রয়োজন হলে এটি কনভার্ট করুন (যেমন KB, MB)।  
- ক্যাশিং বা ভার্সনিং লজিকের জন্য সর্বশেষ পরিবর্তনের তারিখ যাচাই করতে স্ট্যাটস ব্যবহার করুন।
</I18nBangla>
