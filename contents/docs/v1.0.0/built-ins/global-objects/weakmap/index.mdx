# <I18nEnglish>WeakMap</I18nEnglish>
# <I18nBangla>দুর্বল_মানচিত্র</I18nBangla>


<I18nEnglish>
The **WeakMap** object holds key-value pairs where keys must be objects, and values can be any data.  
Keys are held weakly, meaning they do not prevent garbage collection.
</I18nEnglish>

<I18nBangla>
**দুর্বল_মানচিত্র** key-value জোড়া ধরে রাখে যেখানে key অবজেক্ট হতে হবে, আর value যেকোনো ডেটা হতে পারে।  
Keys গুলি দুর্বলভাবে সংরক্ষিত হয়, তাই garbage collection এ মুছে যেতে পারে।
</I18nBangla>

<I18nEnglish>
Create a WeakMap instance using `new WeakMap()` or its Bnlang equivalents.

<I18nBangla>
`new দুর্বল_মানচিত্র()` ব্যবহার করে নতুন WeakMap তৈরি করা যায়।
</I18nBangla>



```bnl
const wm1 = new WeakMap();
const wm2 = new দুর্বল_মানচিত্র();
```

## Methods

### set()

<I18nEnglish>Associates a value with an object key.</I18nEnglish>  
<I18nBangla>একটি object key এর সাথে value সংযুক্ত করে।</I18nBangla>  


```bnl
const wm = new দুর্বল_মানচিত্র();
const key = {};
wm.স্থাপন(key, 123);
```

### get()

<I18nEnglish>Retrieves the value associated with the object key.</I18nEnglish>  
<I18nBangla>object key এর সাথে সংযুক্ত value ফেরত দেয়।</I18nBangla>  


```bnl
console.log(wm.গ্রহন(key)); // 123
```

### has()

<I18nEnglish>Checks if a key exists in the WeakMap.</I18nEnglish>  
<I18nBangla>WeakMap এ key আছে কি না তা যাচাই করে।</I18nBangla>  


```bnl
console.log(wm.আছে(key)); // true
```

### delete()

<I18nEnglish>Removes a key-value pair from the WeakMap.</I18nEnglish>  
<I18nBangla>WeakMap থেকে একটি key-value জোড়া মুছে ফেলে।</I18nBangla>  


```bnl
wm.মুছুন(key);
```

## Limitations

<I18nEnglish>WeakMaps are not enumerable; they have no keys(), values(), entries(), or size.</I18nEnglish>  
<I18nBangla>WeakMap গুলো enumerable নয়; এগুলোর keys(), values(), entries(), বা size নেই।</I18nBangla>
