# <I18nEnglish>string_decoder</I18nEnglish>
# <I18nBangla>string_decoder</I18nBangla>


<I18nEnglish>
The **`string_decoder`** module provides a way to decode `Buffer` objects into strings without breaking multibyte characters. Useful when streaming text data.
</I18nEnglish>

<I18nBangla>
**`string_decoder`** মডিউল `Buffer` কে স্ট্রিং এ ডিকোড করে, মাল্টিবাইট ক্যারেক্টার না ভেঙে। টেক্সট স্ট্রিম হ্যান্ডল করতে দরকার।
</I18nBangla>



---

## API Surface

<I18nEnglish>

### Class: StringDecoder
- **`new StringDecoder([encoding])`** — default `utf8`.  
- **`.write(buffer)`** — returns string decoded from buffer.  
- **`.end([buffer])`** — flush remaining input and end decoding.  

Supported encodings: `utf8`, `utf16le`, `base64`.  

</I18nEnglish>

<I18nBangla>
### ক্লাস
- **`StringDecoder`** — ডিফল্ট `utf8`।  
- **`.write`**, **`.end`** — বাফার থেকে স্ট্রিং এ কনভার্ট।  
- সাপোর্টেড এনকোডিং: `utf8`, `utf16le`, `base64`।  
</I18nBangla>

```bnl
const { StringDecoder } = require("string_decoder");
const decoder = new StringDecoder("utf8");

const buf1 = Buffer.from([0xe0, 0xa4, 0xa8]); // incomplete UTF-8 char
const buf2 = Buffer.from([0xa6]);

console.log(decoder.write(buf1)); // empty, waiting for completion
console.log(decoder.write(buf2)); // outputs proper char (e.g., 'न')

// End flush
console.log(decoder.end());
```

## Notes

<I18nEnglish>
- Unlike `buf.toString()`, StringDecoder preserves whole characters across chunk boundaries.  
- Useful when reading text from streams in arbitrary-sized chunks.  
- Always call `.end()` at the end of input to flush buffered bytes.  
</I18nEnglish>

<I18nBangla>
- `buf.toString()` এর বিপরীতে, StringDecoder ক্যারেক্টার ভাঙে না।  
- স্ট্রিম থেকে টেক্সট পড়ার সময় উপযোগী।  
- সবশেষে `.end()` দিয়ে ফ্লাশ করুন।  
</I18nBangla>
