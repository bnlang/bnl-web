# <I18nEnglish>debugger Statement</I18nEnglish>
# <I18nBangla>ডিবাগার বিবৃতি</I18nBangla>


<I18nEnglish>
The **`debugger;`** statement programmatically triggers a **breakpoint**.  
In browsers, it pauses execution **when DevTools are open**; otherwise it’s ignored.  
In Node.js, it pauses **when an inspector is active** (e.g., running with `--inspect` or an attached debugger).
</I18nEnglish>

<I18nBangla>
**`ডিবাগার;`** দিলে প্রোগ্রাম **ব্রেকপয়েন্টে থামে**।  
ব্রাউজারে **DevTools খোলা থাকলে** থামে; নাহলে উপেক্ষিত হয়।  
Node.js‑এ **ইনস্পেক্টর চালু থাকলে** থামে (যেমন `--inspect` দিয়ে বা ডিবাগার অ্যাটাচ করা হলে)。
</I18nBangla>

// 1) Basic — pauses when DevTools/inspector is active
function mul(a, b) {
  const out = a * b;
  debugger;               // pause here
  return out;
}
mul(6, 7);

// 2) Conditional breakpoint
function evaluate(xs) {
  for (const x of xs) {
    if (x < 0) debugger;
    console.log("x:", x);
  }
}

async function load() {
  const res = await fetch("/data");
  debugger;

const DEBUG = true;
function test() {
  if (DEBUG) debugger;

## Notes

<I18nEnglish>
- `debugger` works in both **sloppy** and **strict** mode; it’s **ignored** when no debugger is attached/open.  
- Prefer **DevTools breakpoints** (including **conditional breakpoints**) for ad‑hoc debugging; keep `debugger` for spots you want checked‑in during development.  
- **Do not ship** `debugger` to production builds — wrap with a flag or strip during bundling.  
- When paused, you can inspect scopes, step through, evaluate expressions, and set **watch**/breakpoints.  
- Language aliases: `ডিবাগার` / `dibagar`.
</I18nEnglish>

<I18nBangla>
- `ডিবাগার` **sloppy/strict** উভয় মোডে কাজ করে; ডিবাগার না থাকলে **উপেক্ষা** হয়।  
- হাতেকলমে ডিবাগে DevTools‑এর **ব্রেকপয়েন্ট/কন্ডিশনাল ব্রেকপয়েন্ট** ব্যবহার করুন; সোর্সে নির্দিষ্ট জায়গায় দরকার হলে `ডিবাগার` রাখুন।  
- **প্রোডাকশনে পাঠাবেন না**—ফ্ল্যাগ দিয়ে গার্ড করুন বা bundling‑এ strip করুন।  
- pause‑এ থাকলে scope দেখে, স্টেপ করতে, এক্সপ্রেশন রান করতে, **watch**/breakpoint সেট করতে পারবেন।  
- ভাষা উপনাম: `ডিবাগার` / `dibagar`.
</I18nBangla>
