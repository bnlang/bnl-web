# <I18nEnglish>Discover JavaScript Timers</I18nEnglish>
# <I18nBangla>জাভাস্ক্রিপ্ট টাইমার আবিষ্কার</I18nBangla>

<I18nEnglish>
Timers in **Bnlang** allow scheduling code to run later or repeatedly.  
The two most common functions are **`setTimeout`** (run once after a delay) and **`setInterval`** (run repeatedly with a fixed delay).  
Timers are essential for animations, retries, scheduled updates, and asynchronous operations.
</I18nEnglish>

<I18nBangla>
**Bnlang** এ টাইমার ব্যবহার করে নির্দিষ্ট সময় পরে বা নির্দিষ্ট বিরতিতে কোড চালানো যায়।  
সবচেয়ে বেশি ব্যবহৃত দুটি ফাংশন হলো **`setTimeout`** (একবার চালায়) এবং **`setInterval`** (বারবার চালায়)।  
অ্যানিমেশন, পুনরায় চেষ্টা, শিডিউল আপডেট এবং অ্যাসিঙ্ক্রোনাস কাজের জন্য টাইমার অত্যন্ত গুরুত্বপূর্ণ।
</I18nBangla>

## setTimeout

<I18nEnglish>
`setTimeout` runs code once after the specified delay (in milliseconds).
</I18nEnglish>

## setInterval

<I18nEnglish>
`setInterval` runs code repeatedly at the specified interval (in milliseconds). Use `clearInterval` to stop it.
</I18nEnglish>

<I18nBangla>
`setInterval` নির্দিষ্ট বিরতিতে (মিলিসেকেন্ডে) কোড বারবার চালায়। এটি থামাতে `clearInterval` ব্যবহার করুন।
</I18nBangla>

## Clearing Timers

<I18nEnglish>
- Use `clearTimeout(timerId)` to cancel a timeout.  
- Use `clearInterval(intervalId)` to cancel an interval.  
Always clear timers when they are no longer needed to avoid memory leaks.
</I18nEnglish>

<I18nBangla>
- `clearTimeout(timerId)` দিয়ে সময়সীমা_নির্ধারণ বন্ধ করুন।  
- `clearInterval(intervalId)` দিয়ে ইন্টারভ্যাল বন্ধ করুন।  
প্রয়োজন শেষ হলে টাইমার ক্লিয়ার করুন যাতে মেমোরি লিক না হয়।
</I18nBangla>

## Best Practices

<I18nEnglish>
- Avoid unnecessary timers; prefer event-driven code when possible.  
- Always clear intervals and timeouts that you set.  
- Be mindful of long-running timers in background tasks.
</I18nEnglish>

<I18nBangla>
- অযথা টাইমার ব্যবহার করবেন না; সম্ভব হলে ইভেন্ট-ড্রিভেন কোড ব্যবহার করুন।  
- সেট করা সময়সীমা_নির্ধারণ বা ইন্টারভ্যাল অবশ্যই ক্লিয়ার করুন।  
- ব্যাকগ্রাউন্ড টাস্কে দীর্ঘ টাইমার ব্যবহার করার সময় সতর্ক থাকুন।
</I18nBangla>
