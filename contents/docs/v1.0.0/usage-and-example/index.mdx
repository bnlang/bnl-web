Welcome to the **Usage and Examples** page of Bnlang.

## Writing your first program

```bnl
function main() {
  print("Hello, World!");
}
```

Run it with:

```bash
bnl main.bnl
```

## Using variables and types

```bnl
let name = "Mamun";
let age = 25;
let isDeveloper = true;

print("Name:", name);
print("Age:", age);
print("Developer?", isDeveloper);
```

## Control flow

```bnl
let score = 85;

if (score >= 80) {
  print("Grade: A+");
} else if (score >= 70) {
  print("Grade: A");
} else {
  print("Grade: Below A");
}
```

## Loops

```bnl
for (let i = 1; i <= 5; i++) {
  print("Number:", i);
}
```

## Functions

```bnl
function greet(user) {
  return "Hello, " + user + "!";
}

print(greet("Mamun"));
```

## Using modules

```bnl
const fs = require('fs');

let data = fs.readFileSync("notes.txt");
print("File content:", data);
```

## আপনার প্রথম প্রোগ্রাম

```bnl
ফাংশন main() {
  লিখুন("হ্যালো, বিশ্ব!");
}
```

চালাতে ব্যবহার করুন:

```bash
bnl main.bnl
```

## ভ্যারিয়েবল ও টাইপ

```bnl
ধরি নাম = "মামুন";
ধরি বয়স = 25;
ধরি ডেভেলপার = true;

লিখুন("নাম:", নাম);
লিখুন("বয়স:", বয়স);
লিখুন("ডেভেলপার?", ডেভেলপার);
```

## শর্তাধারী (Control Flow)

```bnl
ধরি নম্বর = 85;

যদি (নম্বর >= 80) {
  লিখুন("গ্রেড: A+");
} নাহলে যদি (নম্বর >= 70) {
  লিখুন("গ্রেড: A");
} নাহলে {
  লিখুন("গ্রেড: নিচে A");
}
```

## লুপ

```bnl
প্রতি (ধরি i = 1; i <= 5; i++) {
  লিখুন("সংখ্যা:", i);
}
```

## ফাংশন

```bnl
ফাংশন অভিবাদন(ব্যবহারকারী) {
  ফেরত "হ্যালো, " + ব্যবহারকারী + "!";
}

লিখুন(অভিবাদন("মামুন"));
```

## মডিউল ব্যবহার

```bnl
ধ্রুবক fs = প্রয়োজন("fs");

let data = fs.readFileSync("notes.txt");
লিখুন("ফাইল কনটেন্ট:", data);
```
