# <I18nEnglish>Symbol</I18nEnglish>
# <I18nBangla>প্রতীক</I18nBangla>


<I18nEnglish>
The **Symbol** function creates unique values. In Bnlang, you can also use **`প্রতীক`** or **`Protik`**. Symbols are often used as unique object property keys and for special well-known behaviors.
</I18nEnglish>

<I18nBangla>
**প্রতীক** ব্যবহার করে অনন্য মান তৈরি করা যায়। এগুলোকে সাধারণত অবজেক্টের প্রোপার্টি কী এবং বিশেষ বিল্ট-ইন আচরণের জন্য ব্যবহার করা হয়।
</I18nBangla>

- `Symbol.iterator`
- `Symbol.asyncIterator`

- `Symbol.split`
- `Symbol.toStringTag`
- `Symbol.hasInstance`
- `Symbol.isConcatSpreadable`
