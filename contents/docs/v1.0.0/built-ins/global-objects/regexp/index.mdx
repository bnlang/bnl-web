# <I18nEnglish>RegExp</I18nEnglish>
# <I18nBangla>রেজেক্স</I18nBangla>


<I18nEnglish>
Regular expressions are used for pattern matching in strings.  
Constructor names: **`RegExp`**, **`রেজেক্স`**, **`RegEx`**.  
Prototype methods include `test`, `exec`, and symbol-based match helpers.
</I18nEnglish>

<I18nBangla>
স্ট্রিং প্যাটার্ন মেলানোর জন্য রেজেক্স ব্যবহার করা হয়।  
কন্সট্রাক্টর নাম: **`RegExp`**, **`রেজেক্স`**, **`RegEx`**।  
মেথডসমূহের মধ্যে আছে `পরীক্ষা`, `কার্যকর`, এবং প্রতীক ভিত্তিক ম্যাচ ফাংশন।
</I18nBangla>

## Symbol-based methods

<I18nEnglish>
- **match**: `r[Symbol.match](str)` → find matches  
- **matchAll**: `r[Symbol.matchAll](str)` → iterator of all matches  
- **replace**: `r[Symbol.replace](str, replacer)` → replace text  
- **search**: `r[Symbol.search](str)` → find index  
- **split**: `r[Symbol.split](str)` → split string
</I18nEnglish>

<I18nBangla>
- **ম্যাচ**: `r[Symbol.match](স্ট্রিং)` → ম্যাচ খুঁজুন  
- **সব ম্যাচ**: `r[Symbol.matchAll](স্ট্রিং)` → সব ম্যাচের ইটারেটর  
- **বদলাও**: `r[Symbol.replace](স্ট্রিং, প্রতিস্থাপন)` → টেক্সট পরিবর্তন  
- **অনুসন্ধান**: `r[Symbol.search](স্ট্রিং)` → সূচক খুঁজুন  
- **বিভাজন**: `r[Symbol.split](স্ট্রিং)` → স্ট্রিং বিভক্ত করুন
</I18nBangla>
