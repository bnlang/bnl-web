# <I18nEnglish>for Loop</I18nEnglish>
# <I18nBangla>প্রতি লুপ</I18nBangla>


<I18nEnglish>
The **`for`** loop repeats a block of code a specific number of times, or iterates over collections with **`for...of`** and **`for...in`**.  
Use **`break`** to exit early and **`continue`** to skip the current iteration.
</I18nEnglish>

<I18nBangla>
**`প্রতি`** (for) লুপ নির্দিষ্ট সংখ্যকবার কোড চালায়; আবার **`for...of`** দিয়ে iterable (array, string ইত্যাদি) ঘোরা যায় এবং **`for...in`** দিয়ে অবজেক্টের কী ঘোরা যায়।  
আগেভাগে বের হতে **`থামুন`** এবং চলতি ধাপ স্কিপ করতে **`চলুন`** ব্যবহার করুন।
</I18nBangla>



---

## Syntax

<I18nEnglish>

```bnl
// Counting loop
for (let i = start; i < end; i++) {
  // body
}

// Iterate values from an iterable
for (const item of iterable) {
  // use item
}

// Iterate keys from an object
for (const key in object) {
  // use key, object[key]
}
```
</I18nEnglish>

<I18nBangla>

```bnl
// গণনাভিত্তিক লুপ
প্রতি (let i = শুরু; i < শেষ; i++) {
  // কাজ
}

// iterable থেকে মান ঘোরা
প্রতি (const আইটেম এর iterable) {
  // আইটেম ব্যবহার
}

// অবজেক্টের কী ঘোরা
প্রতি (const কী মধ্যে অবজেক্ট) {
  // কী এবং অবজেক্ট[কী]
}
```
</I18nBangla>

```bnl
// 1) Counting with break / continue
let sum = 0;
for (let i = 1; i <= 10; i++) {
  if (i === 5) continue;         // skip 5
  if (i > 8) break;               // stop at 9
  sum += i;
}
console.log("sum =", sum);

// 2) for...of over arrays and strings
const nums = [2, 4, 6];
for (const n of nums) {
  print(n * n);
}

for (const ch of "ABC") {
  console.log(ch);
}

// 3) for...in over object keys
const user = { name: "Rafi", city: "Dhaka" };
for (const k in user) {
  console.info(k, "=>", user[k]);
}
```

## Notes

<I18nEnglish>
- Prefer `let` in loop headers so the index is block‑scoped.  
- Use **`for...of`** for array/iterable values; use **`for...in`** for object keys.  
- Avoid using `for...in` on arrays (order is not guaranteed).  
- Combine with **`break`** (exit) and **`continue`** (skip) when needed.
</I18nEnglish>

<I18nBangla>
- লুপ হেডারে `let` ব্যবহার করুন যাতে সূচকটি ব্লক‑স্কোপড থাকে।  
- অ্যারে/ইটারেবল মানের জন্য **`for...of`**, অবজেক্টের কী‑এর জন্য **`for...in`**।  
- অ্যারেতে **`for...in`** এড়িয়ে চলুন (অর্ডার নির্ভরযোগ্য নয়)।  
- প্রয়োজনে **`থামুন`** (break) ও **`চলুন`** (continue) ব্যবহার করুন।
</I18nBangla>
