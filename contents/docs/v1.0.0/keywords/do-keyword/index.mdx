# <I18nEnglish>do...while Loop</I18nEnglish>
# <I18nBangla>করুন...যতক্ষণ লুপ</I18nBangla>


<I18nEnglish>
The **`do...while`** loop runs the block **at least once**, then repeats while the condition remains **true**.  
The condition is checked **after** each iteration. Use **`break`** to exit early and **`continue`** to skip to the next check.
</I18nEnglish>

<I18nBangla>
**`করুন...যতক্ষণ`** লুপে ব্লকটি **অন্তত একবার** চালানো হয়, তারপর শর্ত **সত্য** থাকা পর্যন্ত পুনরাবৃত্তি হয়।  
প্রতিটি ইটারেশনের **পরে** শর্ত পরীক্ষা করা হয়। আগেভাগে বের হতে **`থামুন`** এবং চলতি ধাপ স্কিপ করতে **`চলুন`** ব্যবহার করুন।
</I18nBangla>



---

## Syntax

<I18nEnglish>

```bnl
do {
  // body
} while (condition);
```
</I18nEnglish>

<I18nBangla>

```bnl
করুন {
  // কাজ
} যতক্ষণ (শর্ত);
```
</I18nBangla>

```bnl
// 1) Runs at least once
let i = 10;
do {
  console.log("i =", i); // will run once even if condition is false
  i++;
} while (i < 5);

// 2) Iterate an array with an index
const nums = [2, 4, 6];
let idx = 0;
do {
  print(nums[idx]);
  idx++;
} while (idx < nums.length);

// 3) continue / break behavior
let n = 0;
do {
  n++;
  if (n % 2 === 0) continue; // skip evens
  console.info("odd:", n);
  if (n >= 7) break;         // exit early
} while (n < 10);
```

## Notes

<I18nEnglish>
- The body executes **before** the condition check, so it always runs at least once.  
- Don’t forget the **semicolon** (`;`) after the closing `while (condition)`.  
- `continue` jumps to the **condition check**; `break` exits the loop.  
- Prefer `while`/`for` unless you specifically need “run once, then check” semantics.
</I18nEnglish>

<I18nBangla>
- বডি **শর্তের আগে** চালানো হয়—তাই অন্তত একবার চলবেই।  
- `while (শর্ত)`-এর পরে **সেমিকোলন** (`;`) দিতে ভুলবেন না।  
- **`চলুন`** দিলে পরের **শর্ত যাচাই** তে লাফ দেবে; **`থামুন`** দিলে লুপ থেকে বের হবে।  
- “একবার চালাও, তারপর চেক করুন”—এই প্রয়োজন না হলে `যতক্ষণ`/`প্রতি` (while/for) যথেষ্ট।
</I18nBangla>
