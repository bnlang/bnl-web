# <I18nEnglish>Reflect</I18nEnglish>
# <I18nBangla>প্রতিফলন</I18nBangla>


<I18nEnglish>
**`Reflect`** provides functions for object meta-operations.  
Available namespaces: **`Reflect`**, **`প্রতিফলন`**, **`Protifolon`**.
</I18nEnglish>

<I18nBangla>
**`প্রতিফলন`** অবজেক্ট মেটা-অপারেশন করার জন্য ব্যবহার হয়।  
নেমস্পেস: **`Reflect`**, **`প্রতিফলন`**, **`Protifolon`**।
</I18nBangla>

## Common Methods

<I18nEnglish>
- `Reflect.get(obj, key)` → Bangla: `প্রতিফলন.গ্রহন(obj, key)` → Banglish: `Protifolon.grohon(obj, key)`  
- `Reflect.set(obj, key, value)` → Bangla: `প্রতিফলন.স্থাপন(obj, key, value)` → Banglish: `Protifolon.sthapan(obj, key, value)`  
- `Reflect.apply(fn, thisArg, args)` → Bangla: `প্রতিফলন.প্রয়োগ(fn, thisArg, args)` → Banglish: `Protifolon.proyog(fn, thisArg, args)`  
- `Reflect.construct(Ctor, args)` → Bangla: `প্রতিফলন.নির্মাণ(Ctor, args)` → Banglish: `Protifolon.nirman(Ctor, args)`  
</I18nEnglish>

<I18nBangla>
- `প্রতিফলন.গ্রহন(obj, key)` → অবজেক্ট থেকে মান নেওয়া  
- `প্রতিফলন.স্থাপন(obj, key, value)` → অবজেক্টে মান বসানো  
- `প্রতিফলন.প্রয়োগ(fn, thisArg, args)` → ফাংশন কল  
- `প্রতিফলন.নির্মাণ(Ctor, args)` → নতুন অবজেক্ট তৈরি  
</I18nBangla>

```bnl
const obj = { x: 10 };

// get & set
console.log(Reflect.get(obj, "x")); // 10
Reflect.set(obj, "y", 20);
console.log(obj.y); // 20

// construct
function Point(x, y) { this.x = x; this.y = y; }
const p = Reflect.construct(Point, [1, 2]);
console.log(p); // Point { x: 1, y: 2 }
```
