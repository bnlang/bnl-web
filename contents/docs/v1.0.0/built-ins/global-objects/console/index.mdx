## Logging Methods

<I18nEnglish>
- `console.log("...")` → general logging  
- `console.info("...")` → info log  
- `console.debug("...")` → debug log  
- `console.warn("...")` → warning  
- `console.error("...")` → error  
- `console.success("...")` → success message
</I18nEnglish>

<I18nBangla>
- `কনসোল.লগ("...")` → সাধারণ লগ  
- `কনসোল.তথ্য("...")` → তথ্য লগ  
- `কনসোল.ডিবাগ("...")` → ডিবাগ লগ  
- `কনসোল.সতর্ক("...")` → সতর্কবার্তা  
- `কনসোল.ত্রুটি("...")` → ত্রুটির বার্তা  
- `কনসোল.সাফল্য("...")` → সফলতার বার্তা
</I18nBangla>

<I18nBanglish>
- `console.log("...")` → normal log  
- `console.tothyo("...")` → info log  
- `console.dibag("...")` → debug log  
- `console.sotorko("...")` → warning  
- `console.truti("...")` → error  
- `console.safollo("...")` → success
</I18nBanglish>

## Extra Features

<I18nEnglish>
- `console.table(data)` → pretty table output  
- `console.time("label")` / `console.timeEnd("label")` → measure execution time  
- `console.count("label")` → count occurrences  
- `console.group()` / `console.groupEnd()` → grouped logs  
- `console.json(obj)` → pretty-print JSON
</I18nEnglish>

<I18nBangla>
- `কনসোল.টেবিল(ডাটা)` → সুন্দর টেবিল আউটপুট  
- `কনসোল.সময়("label")` / `কনসোল.সময়শেষ("label")` → সময় পরিমাপ  
- `কনসোল.গণনা("label")` → গণনা সংখ্যা  
- `কনসোল.দল()` / `কনসোল.দলশেষ()` → গ্রুপড লগ  
- `কনসোল.জেসন(obj)` → JSON প্রিন্ট
</I18nBangla>

<I18nBanglish>
- `console.table(data)` → table output  
- `console.shomoy("label")` / `console.shomoyShesh("label")` → time measure  
- `console.gonona("label")` → count  
- `console.dol()` / `console.dolShesh()` → grouped logs  
- `console.json(obj)` → JSON print
</I18nBanglish>
