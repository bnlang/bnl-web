# <I18nEnglish>Map</I18nEnglish>
# <I18nBangla>মানচিত্র</I18nBangla>


<I18nEnglish>
Bnlang provides the **`Map`** type also as **`মানচিত্র`** and **`Manchitro`**.  
It stores key-value pairs and remembers insertion order.
</I18nEnglish>

<I18nBangla>
Bnlang এ **`Map`**, **`মানচিত্র`**, **`Manchitro`** নামে ব্যবহার করা যায়।  
এটি key-value জোড়া সংরক্ষণ করে এবং ইনসারশনের ক্রম মনে রাখে।
</I18nBangla>

## Methods

<I18nEnglish>
- Set value: `m.set(k,v)` → `m.স্থাপন(k,v)` → `m.sthapon(k,v)`  
- Get value: `m.get(k)` → `m.গ্রহন(k)` → `m.grohon(k)`  
- Delete: `m.delete(k)` → `m.মুছুন(k)` → `m.muchun(k)`  
- Clear: `m.clear()` → `m.পরিষ্কার()` → `m.porishkar()`  
- Has key: `m.has(k)` → `m.আছে(k)` → `m.ache(k)`  
- Keys: `m.keys()` → `m.চাবিগুলী()` → `m.chabiguli()`  
- Values: `m.values()` → `m.মানগুলী()` → `m.managuli()`  
- Entries: `m.entries()` → `m.জোড়াগুলি()` → `m.joraguli()`  
- For each: `m.forEach(fn)` → `m.প্রতিটি_চালান(fn)` → `m.protitiCalan(fn)`  
- Size: `m.size` → `m.আকার` → `m.akar`
</I18nEnglish>

<I18nBangla>
- ভ্যালু সেট: `m.স্থাপন(k,v)`  
- ভ্যালু নেয়া: `m.গ্রহন(k)`  
- মুছা: `m.মুছুন(k)`  
- পরিষ্কার করা: `m.পরিষ্কার()`  
- কী আছে?: `m.আছে(k)`  
- চাবি: `m.চাবিগুলী()`  
- মান: `m.মানগুলী()`  
- জোড়া: `m.জোড়াগুলি()`  
- প্রতিটি চালানো: `m.প্রতিটি_চালান(fn)`  
- আকার: `m.আকার`
</I18nBangla>
