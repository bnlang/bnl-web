# <I18nEnglish>JSON</I18nEnglish>
# <I18nBangla>জেসন</I18nBangla>


<I18nEnglish>
Bnlang provides the JSON object in three forms: **`JSON`**, **`জেসন`**, **`Jeson`**.  
Use it to parse JSON text into objects and to stringify objects into JSON text.
</I18nEnglish>

<I18nBangla>
Bnlang এ JSON পাওয়া যায় **`JSON`**, **`জেসন`**, **`Jeson`** নামে।  
এটি ব্যবহার করে JSON টেক্সট থেকে অবজেক্ট বানানো যায় এবং অবজেক্টকে JSON টেক্সটে রূপান্তর করা যায়।
</I18nBangla>



---

## Methods

<I18nEnglish>
- Parse JSON string: `JSON.parse(text)` → `জেসন.বিশ্লেষণ(text)` → `Jeson.bisleshon(text)`  
- Stringify object: `JSON.stringify(obj)` → `জেসন.অক্ষরায়ন(obj)` → `Jeson.okkhorayan(obj)`
</I18nEnglish>

<I18nBangla>
- JSON স্ট্রিং বিশ্লেষণ: `জেসন.বিশ্লেষণ(text)` / `JSON.parse(text)` / `Jeson.bisleshon(text)`  
- অবজেক্টকে JSON টেক্সটে রূপান্তর: `জেসন.অক্ষরায়ন(obj)` / `JSON.stringify(obj)` / `Jeson.okkhorayan(obj)`
</I18nBangla>
