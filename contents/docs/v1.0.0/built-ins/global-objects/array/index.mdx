# <I18nEnglish>Array</I18nEnglish>
# <I18nBangla>তালিকা</I18nBangla>
# <I18nBanglish>Array</I18nBanglish>

<I18nEnglish>
In Bnlang, `Array` works like in JavaScript but can also be used with Bangla and Banglish names.  
Available as **`Array`**, **`তালিকা`**, and **`Talika`**.  
Use these for creating and working with lists of values.
</I18nEnglish>

<I18nBangla>
Bnlang এ `Array` JavaScript এর মত কাজ করে তবে বাংলায় এবং Banglish নামেও ব্যবহার করা যায়।  
এটি পাওয়া যায় **`Array`**, **`তালিকা`**, এবং **`Talika`** হিসেবে।  
এগুলো দিয়ে মানের তালিকা তৈরি ও ব্যবহার করা যায়।
</I18nBangla>

<I18nBanglish>
Bnlang e `Array` JavaScript er moto kaj kore kintu Bangla ar Banglish name o use kora jay.  
Eta deya ache **`Array`**, **`তালিকা`**, ar **`Talika`** hisebe.  
Egulo diye list toiri o kaj kora jay.
</I18nBanglish>

## Static Methods

<I18nEnglish>
- `Array.isArray()` → also available as `তালিকা.তালিকা_কি`, `Talika.talikaKi`  
- `Array.from()` → also available as `তালিকা.থেকে`, `Talika.theke`  
- `Array.fromAsync()` → also available as `তালিকা.অসমলয়_থেকে`, `Talika.osomoloyTheke`  
- `Array.of()` → also available as `তালিকা.এর`, `Talika.er`
</I18nEnglish>

<I18nBangla>
- `Array.isArray()` → ব্যবহার করা যায় `তালিকা.তালিকা_কি`  
- `Array.from()` → ব্যবহার করা যায় `তালিকা.থেকে`  
- `Array.fromAsync()` → ব্যবহার করা যায় `তালিকা.অসমলয়_থেকে`  
- `Array.of()` → ব্যবহার করা যায় `তালিকা.এর`
</I18nBangla>

<I18nBanglish>
- `Array.isArray()` → use kora jay `Talika.talikaKi` name o  
- `Array.from()` → use kora jay `Talika.theke` name o  
- `Array.fromAsync()` → use kora jay `Talika.osomoloyTheke` name o  
- `Array.of()` → use kora jay `Talika.er` name o
</I18nBanglish>

## Prototype Methods (Common)

<I18nEnglish>
- Add/Remove → `push()` (`শেষে_যোগ`, `shesheJog`), `pop()` (`শেষ_উপাদান`, `sheshUpadan`)  
- Shift/Unshift → `shift()` (`প্রথম_উপাদান`, `prothomUpadan`), `unshift()` (`প্রথমে_যোগ`, `prothomeJog`)  
- Map/Filter → `map()` (`মাপ`, `map`), `filter()` (`বাছাই`, `bachai`)  
- Reduce → `reduce()` (`কমান`, `koman`), `reduceRight()` (`ডান_থেকে_কমান`, `danThekeKoman`)  
- ForEach → `forEach()` (`প্রতিটি_চালাও`, `protitiCalao`)
</I18nEnglish>

<I18nBangla>
- যোগ/অপসারণ → `শেষে_যোগ()`, `শেষ_উপাদান()`  
- প্রথম উপাদান → `প্রথম_উপাদান()`, `প্রথমে_যোগ()`  
- রূপান্তর → `মাপ()`, `বাছাই()`  
- যোগফল → `কমান()`, `ডান_থেকে_কমান()`  
- প্রতিটি চালানো → `প্রতিটি_চালাও()`
</I18nBangla>

<I18nBanglish>
- Add/Remove → `shesheJog()`, `sheshUpadan()`  
- First element → `prothomUpadan()`, `prothomeJog()`  
- Transform → `map()`, `bachai()`  
- Reduce → `koman()`, `danThekeKoman()`  
- For each → `protitiCalao()`
</I18nBanglish>
