# <I18nEnglish>continue Keyword</I18nEnglish>
# <I18nBangla>চলুন কীওয়ার্ড</I18nBangla>


<I18nEnglish>
The **`continue`** statement skips the rest of the **current** iteration of a loop and moves to the next check/iteration.  
Use it in `for`, `while`, and `do...while` loops to ignore certain cases without exiting the loop.
</I18nEnglish>

<I18nBangla>
**`চলুন`** স্টেটমেন্ট বর্তমান ইটারেশনের বাকি অংশ **স্কিপ** করে পরবর্তী ইটারেশনে চলে যায়।  
`প্রতি` (for), `যতক্ষণ` (while), এবং `করো...যতক্ষণ` (do...while) লুপে কিছু কেস বাদ দিতে এটি ব্যবহার করুন।
</I18nBangla>

// 1) Skip even numbers
let sum = 0;
for (let i = 1; i <= 10; i++) {
  if (i % 2 === 0) continue;
  sum += i;
}
console.log("Odd sum =", sum);

// 2) while input validation
let n = 0;
while (n < 8) {
  n++;
  if (n < 3) continue;       // skip 1,2
  console.info("processing:", n);
}

// 3) for...of — skip empty strings
const words = ["Hi", "", "Bnlang", " ", "OK"];
for (const w of words) {
  if (w.trim().length === 0) continue;
  print("=>", w);
}

## Notes

<I18nEnglish>
- `continue` does **not** exit the loop; it only skips to the next iteration.  
- In `while`/`do...while`, ensure state updates happen **before** `continue` to avoid infinite loops.  
- Prefer clear conditions; overusing `continue` can hurt readability. Use filtering (`.filter`) when appropriate.
</I18nEnglish>

<I18nBangla>
- **`চলুন`** লুপ থেকে বের করে না; কেবল পরের ইটারেশনে লাফ দেয়।  
- `যতক্ষণ`/`করো...যতক্ষণ` লুপে **স্টেট আপডেট** `চলুন`-র আগেই করুন, নাহলে অনন্ত লুপ হতে পারে।  
- পড়তে সহজ রাখতে অযথা `চলুন` ব্যবহার করবেন না; প্রয়োজনে ফিল্টারিং (`.filter`) ব্যবহার করুন।
</I18nBangla>
