# <I18nEnglish>try–catch–finally</I18nEnglish>
# <I18nBangla>চেষ্টা–ধরুন–অবশেষে</I18nBangla>


<I18nEnglish>
Use **`try`** for code that might throw. Handle failures in **`catch (err)`**.  
Use **`finally`** for cleanup that must run whether success or failure.  
Throw errors with **`throw`** to signal exceptional conditions.
</I18nEnglish>

<I18nBangla>
সম্ভাব্য ত্রুটি‑সৃষ্টিকারী কোড **`চেষ্টা`** ব্লকে লিখুন। ব্যর্থতা ধরতে **`ধরুন (ত্রু)`** ব্যবহার করুন।  
সাফল্য‑ব্যর্থতা নির্বিশেষে চলবে এমন ক্লিন‑আপের জন্য **`অবশেষে`** ব্যবহার করুন।  
অসাধারণ পরিস্থিতি জানাতে **`নিক্ষেপ`** দিয়ে ত্রুটি ছুঁড়ুন।
</I18nBangla>

## Syntax

<I18nEnglish>

```bnl
try {
  // risky work
} catch (err) {
  // handle
} finally {
  // cleanup (always runs)
}

// You can omit catch, but then finally must be present
try {
  // ...
} finally {
  // cleanup
}

// Throwing
throw new Error("message");
```
</I18nEnglish>

<I18nBangla>

```bnl
চেষ্টা {
  // ঝুঁকিপূর্ণ কাজ
} ধরুন (ত্রু) {
  // হ্যান্ডলিং
} অবশেষে {
  // ক্লিন‑আপ (সবসময় চলবে)
}

// ধরুন বাদ দিতে পারেন, তবে অবশেষে থাকতে হবে
চেষ্টা {
  // ...
} অবশেষে {
  // ক্লিন‑আপ
}

// নিক্ষেপ
নিক্ষেপ new Error("বার্তা");
```
</I18nBangla>

// 1) JSON parsing
try {
  const data = '{"name":"Bnlang"}';
  const obj = JSON.parse(data);
  console.log("name:", obj.name);
} catch (err) {
  console.error("parse failed:", err.message);
} finally {
  console.info("parse attempt done");
}

// 2) Validate + rethrow
function loadUser(id) {
  try {
    if (id == null) throw new Error("ID required");
    // ... read from DB
    return { id, name: "Mamun" };
  } catch (err) {
    console.error("load failed:", err.message);
    throw err; // rethrow original
  } finally {
    console.log("resource cleanup");
  }
}

// 3) finally still runs after return
function divide(a, b) {
  try {
    if (b === 0) throw new Error("Division by zero");
    return a / b;
  } catch (err) {
    return NaN;
  } finally {
    console.info("division done");
  }
}

// 4) with async/await
async function download(url) {
  try {
    const res = await fetch(url);
    if (!res.ok) throw new Error("Bad response");
    return await res.text();
  } catch (err) {
    console.error("download failed:", err.message);
    return null;
  } finally {
    console.log("network cleanup");
  }
}

## Notes

<I18nEnglish>
- `try` may be paired with **either** `catch` **or** `finally` (or both), but not neither.  
- The `finally` block runs even if there is a `return`, `break`, `continue`, or `throw` inside `try`/`catch`.  
- Prefer **rethrowing** the same error object (`throw err`) to preserve the original stack.  
- For **async** code, `try/catch` works with `await`; for bare Promises, use `.catch(...)`.  
- Avoid swallowing errors silently; log and rethrow when appropriate.
</I18nEnglish>

<I18nBangla>
- `চেষ্টা` ব্লকের সাথে **`ধরুন`** বা **`অবশেষে`**—যেকোনো একটি থাকা চাই (বা দুটোই); কোনোটাই না হলে অবৈধ।  
- `অবশেষে` ব্লক `return`, `break`, `continue`, `নিক্ষেপ` থাকলেও চালানো হয়।  
- একই ত্রুটি অবজেক্ট **আবার নিক্ষেপ** করলে (যেমন `নিক্ষেপ ত্রু`) স্ট্যাক তথ্য বজায় থাকে।  
- **অসমলয়** কোডে `অপেক্ষা` (`await`) ব্যবহার করলে `চেষ্টা/ধরুন` যথারীতি কাজ করে; প্রমিসে `.catch(...)` ব্যবহার করুন।  
- ত্রুটি চুপচাপ গিলে ফেলা এড়িয়ে চলুন—প্রয়োজনে লগ করে **পুনরায় নিক্ষেপ** করুন।
</I18nBangla>
