# <I18nEnglish>Writing Files in Bnlang</I18nEnglish>
# <I18nBangla>Bnlang এ ফাইল লেখা</I18nBangla>


<I18nEnglish>
Writing files is a common task when working with file systems.  
Bnlang provides both **synchronous** and **asynchronous** methods to write data into files.  
Synchronous methods block program execution until the operation is finished, while asynchronous methods allow the program to continue running in the background.
</I18nEnglish>

<I18nBangla>
ফাইল লেখা ফাইল সিস্টেম নিয়ে কাজ করার সময় একটি সাধারণ কাজ।  
Bnlang ফাইল লেখার জন্য **সিঙ্ক্রোনাস** এবং **অ্যাসিঙ্ক্রোনাস** উভয় পদ্ধতি প্রদান করে।  
সিঙ্ক্রোনাস মেথডে অপারেশন শেষ না হওয়া পর্যন্ত প্রোগ্রাম ব্লক থাকে, আর অ্যাসিঙ্ক্রোনাস মেথডে প্রোগ্রাম ব্যাকগ্রাউন্ডে চালু থাকে।
</I18nBangla>

## Example: Asynchronous Write

<Tabs defaultValue="bangla" className="w-full">

```js
const fs = require("fs");

fs.writeFile("example.txt", "Hello World!", (err) => {
  if (err) {
    console.error("Error writing file:", err);
    return;
  }
  console.log("File written asynchronously!");
});
```

## Best Practices

<I18nEnglish>
- Use asynchronous writes in production for better performance.  
- Synchronous writes are suitable for small scripts or setup tasks.  
- Always handle errors when writing files.  
- Use proper encoding like `utf-8` for text files.  
- Be careful when overwriting files; check before writing if necessary.
</I18nEnglish>

<I18nBangla>
- প্রোডাকশনে ভালো পারফরম্যান্সের জন্য অ্যাসিঙ্ক্রোনাস মেথড ব্যবহার করুন।  
- ছোট স্ক্রিপ্ট বা সেটআপ টাস্কের জন্য সিঙ্ক্রোনাস মেথড উপযুক্ত।  
- ফাইল লেখার সময় সর্বদা ত্রুটি হ্যান্ডল করুন।  
- টেক্সট ফাইলের জন্য সঠিক এনকোডিং (যেমন `utf-8`) ব্যবহার করুন।  
- ফাইল ওভাররাইট করার সময় সতর্ক থাকুন; প্রয়োজন হলে আগে যাচাই করুন।
</I18nBangla>
