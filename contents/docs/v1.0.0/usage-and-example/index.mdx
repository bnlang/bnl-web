<I18nEnglish>
Welcome to the **Usage and Examples** page of Bnlang.

## Writing your first program

```bnl
function main() {
  print("Hello, World!");
}
```

Run it with:

```bash
bnl main.bnl
```

## Using variables and types

```bnl
let name = "Mamun";
let age = 25;
let isDeveloper = true;

print("Name:", name);
print("Age:", age);
print("Developer?", isDeveloper);
```

## Control flow

```bnl
let score = 85;

if (score >= 80) {
  print("Grade: A+");
} else if (score >= 70) {
  print("Grade: A");
} else {
  print("Grade: Below A");
}
```

## Loops

```bnl
for (let i = 1; i <= 5; i++) {
  print("Number:", i);
}
```

## Functions

```bnl
function greet(user) {
  return "Hello, " + user + "!";
}

print(greet("Mamun"));
```

## Using modules

```bnl
const fs = require('fs');

let data = fs.readFileSync("notes.txt");
print("File content:", data);
```

## শর্তাধারী (Control Flow)

```bnl
ধরি নম্বর = 85;

  লিখুন("গ্রেড: A+");

  লিখুন("গ্রেড: A");

}
```
